YT Selector 개인정보처리방침

최종 업데이트: 2024년 3월 21일

1. 수집하는 정보

YT Selector는 다음과 같은 정보를 수집합니다:

  • 선택한 YouTube 영상의 제목과 URL
  • 영상 선택 시간

2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 선택한 영상 목록 관리
  • 구글 시트에 선택한 영상 정보 저장
  • 선택한 영상들을 새 탭에서 열기

3. 정보 저장 및 보관

수집된 정보는 다음과 같이 저장됩니다:

  • 로컬 저장소: 선택한 영상 목록을 브라우저의 로컬 저장소에 임시 보관
  • 구글 시트: 사용자가 요청한 경우에만 선택한 영상 정보를 구글 시트에 저장

4. 정보 공유

YT Selector는 수집된 정보를 다음과 같은 경우에만 공유합니다:

  • 사용자가 구글 시트 저장 기능을 사용할 때 구글 시트 API를 통한 저장
  • 법적 요구사항에 따른 경우

5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 선택한 영상 목록 초기화
  • 구글 시트에 저장된 데이터 삭제
  • 확장 프로그램 제거를 통한 모든 데이터 삭제

6. 보안

YT Selector는 다음과 같은 보안 조치를 취합니다:

  • 구글 OAuth2를 통한 안전한 인증
  • 로컬 저장소를 통한 데이터 보관
  • HTTPS를 통한 안전한 데이터 전송
